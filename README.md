# Device Monitoring Dashboard

A React-based dashboard that simulates monitoring laboratory measurement devices.
The application displays device status, measurement values, and allows users to search and filter devices.

This project demonstrates modern frontend development practices including component-based architecture, API simulation, filtering, and responsive UI design.

---

## Live Demo

https://hamzamohsin1996.github.io/Device-Monitoring-Dashboard/

---

## Features

• Display 100 simulated monitoring devices
• Search devices by name
• Filter devices by status (Online / Offline)
• Component-based architecture using React
• Responsive layout using Bootstrap 5
• Simulated API data service
• Status indicators for device availability

## Installation

Clone the repository:

```
git clone https://github.com/HamzaMohsin1996/Device-Monitoring-Dashboard.git
```

Navigate into the project:

```
cd Device-Monitoring-Dashboard
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```

---

## Build for Production

```
npm run build
```
